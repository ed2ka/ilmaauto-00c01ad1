# ILMA AUTO standalone demo

Ovo je čista HTML/CSS/JavaScript verzija bez Reacta, backend-a i eksternih dependency-ja.

## Pokretanje

Otvorite `index.html` direktno u browseru ili uploadujte kompletan sadržaj foldera na cPanel / public_html.

## Stranice

- index.html
- pretraga.html
- dio.html
- prijava.html
- profil.html
- podrska.html
- reset-password.html
- privatnost.html
- uslovi-koristenja.html
- uslovi-kupovine.html
- politika-povrata.html
- kolacici.html
- 404.html

Svi modali, tabovi, FAQ, dropdown meniji, pretraga i demo forme rade lokalno u browseru.
